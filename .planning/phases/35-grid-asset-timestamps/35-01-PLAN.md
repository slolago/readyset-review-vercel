# Plan 35-01: Add upload date/time to AssetCard info row

## Goal
Every grid card shows the upload date and time (e.g. "Apr 14, 3:42 PM") in the info area below the filename — always visible, no hover required. This is the primary way to distinguish two V1 cards that appear after unstacking a version stack.

## Context
- **File**: `src/components/files/AssetCard.tsx`
- **Current info row** (lines 427–461):
  - Line 1: asset name (truncated, with title tooltip)
  - Line 2: `formatBytes(asset.size)` left · comment count + version count right
- **`asset.createdAt`** is a Firestore `Timestamp` — when served from the API it is serialized as `{ _seconds: number, _nanoseconds: number }`, so the `toDate()` method is absent; must use the `_seconds` fallback pattern used throughout the codebase
- `Clock` icon is already imported in the file

## Implementation

### Step 1 — Compute upload date near the top of the component body

Add after the existing `const isUploading` line:

```tsx
const uploadDate: Date | null =
  typeof (asset.createdAt as any)?.toDate === 'function'
    ? (asset.createdAt as any).toDate()
    : (asset.createdAt as any)?._seconds
    ? new Date((asset.createdAt as any)._seconds * 1000)
    : null;

const uploadDateLabel = uploadDate
  ? uploadDate.toLocaleDateString('en-US', { month: 'short', day: 'numeric' }) +
    ', ' +
    uploadDate.toLocaleTimeString('en-US', { hour: 'numeric', minute: '2-digit' })
  : '';
```

### Step 2 — Add date/time row to the info section

In the `{/* Info */}` block, after the existing `flex items-center justify-between mt-0.5` row (the size + badges row), add:

```tsx
{uploadDateLabel && (
  <div className="flex items-center gap-1 mt-0.5">
    <Clock className="w-3 h-3 text-frame-textMuted flex-shrink-0" />
    <p className="text-xs text-frame-textMuted">{uploadDateLabel}</p>
  </div>
)}
```

### Complete resulting info block

```tsx
{/* Info */}
<div className="p-3">
  {isRenaming ? (
    <input ... />
  ) : (
    <p className="text-sm font-medium text-white truncate" title={asset.name}>
      {asset.name}
    </p>
  )}
  <div className="flex items-center justify-between mt-0.5">
    <p className="text-xs text-frame-textMuted">{formatBytes(asset.size)}</p>
    <div className="flex items-center gap-2">
      {commentCount > 0 && (
        <span className="flex items-center gap-0.5 text-xs text-frame-textMuted">
          <MessageSquare className="w-3 h-3" />
          {commentCount > 99 ? '99+' : commentCount}
        </span>
      )}
      {versionCount > 1 && (
        <p className="text-xs text-frame-accent font-medium">{versionCount} versions</p>
      )}
    </div>
  </div>
  {uploadDateLabel && (
    <div className="flex items-center gap-1 mt-0.5">
      <Clock className="w-3 h-3 text-frame-textMuted flex-shrink-0" />
      <p className="text-xs text-frame-textMuted">{uploadDateLabel}</p>
    </div>
  )}
</div>
```
