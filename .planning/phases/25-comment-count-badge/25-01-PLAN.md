# Plan 25-01: Comment Count Badge on AssetCard

## Objective

Add a comment count badge to `AssetCard` in grid view. The badge shows a `MessageSquare` icon plus the count when `_commentCount > 0`. Counts above 99 display as "99+". No badge is shown when count is 0 or absent. No new API calls — reads `asset._commentCount` which is already returned by the assets endpoint.

## Context

- `src/components/files/AssetCard.tsx` — grid card component; info section at bottom has asset name, size, version count. Badge goes in the info section.
- `src/types/index.ts` — `_commentCount?: number` already defined on `Asset`.
- `src/components/files/AssetListView.tsx` — list view shows `_commentCount` as plain number; grid should show badge with icon.
- `src/app/api/assets/route.ts` — already populates `asset._commentCount` from a comment count map.

## Tasks

### Task 1: Add comment count badge to AssetCard info section

**Type:** auto

**Done when:** AssetCard renders a `MessageSquare` icon + count badge in the info section when `_commentCount > 0`, with "99+" cap, and renders nothing when count is 0 or absent.

**Implementation:**

1. Import `MessageSquare` from `lucide-react` in AssetCard.tsx.
2. Read `const commentCount = (asset as any)._commentCount as number | undefined ?? 0;` near the top of the component (alongside `versionCount`).
3. In the info section's `<div className="flex items-center justify-between mt-0.5">`, add a comment badge after the size/version row:
   - Only render when `commentCount > 0`.
   - Display: `<MessageSquare className="w-3 h-3" />` + `{commentCount > 99 ? '99+' : commentCount}`.
   - Style: small pill using existing muted text color, matching the existing version count style.

**Files:** `src/components/files/AssetCard.tsx`

**Verification:**
- `grep -n "MessageSquare" src/components/files/AssetCard.tsx` returns a match
- `grep -n "commentCount" src/components/files/AssetCard.tsx` returns at least 2 matches (declaration + conditional render)
- `grep -n "99+" src/components/files/AssetCard.tsx` returns a match

**Acceptance criteria:**
- Badge renders when `_commentCount > 0`
- No badge when `_commentCount` is 0 or undefined
- Counts > 99 show "99+"
- Uses `MessageSquare` Lucide icon
- No new API calls
