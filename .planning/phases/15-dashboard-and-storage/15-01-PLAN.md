Auth pattern: identical to src/app/api/projects/route.ts — use `getAuthenticatedUser` and `getAdminDb`.

Logic:

1. Auth check — return 401 if no user.

2. Fetch all projects (same query as projects/route.ts GET):
   ```typescript
   const snap = await db.collection('projects').get();
   const userProjects = snap.docs
     .map(d => ({ id: d.id, ...d.data() } as any))
     .filter(p => p.ownerId === user.id || p.collaborators?.some((c: any) => c.userId === user.id));
   const projectIds = userProjects.map((p: any) => p.id);
   ```

3. Deduplicate collaborators — iterate all projects, collect non-self userId values into a Set:
   ```typescript
   const collaboratorSet = new Set<string>();
   for (const p of userProjects) {
     for (const c of (p.collaborators || [])) {
       if (c.userId !== user.id) collaboratorSet.add(c.userId);
     }
   }
   ```

4. Collection group query for assets — chunk projectIds into groups of 30 (Firestore `in` limit):
   ```typescript
   let assetCount = 0;
   let storageBytes = 0;
   if (projectIds.length > 0) {
     const chunks: string[][] = [];
     for (let i = 0; i < projectIds.length; i += 30) chunks.push(projectIds.slice(i, i + 30));
     for (const chunk of chunks) {
       const assetsSnap = await db.collectionGroup('assets')
         .where('projectId', 'in', chunk)
         .get();
       assetCount += assetsSnap.size;
       for (const doc of assetsSnap.docs) {
         storageBytes += (doc.data().size as number) || 0;
       }
     }
   }
   ```

5. Return JSON:
   ```typescript
   return NextResponse.json({
     projectCount: userProjects.length,
     assetCount,
     collaboratorCount: collaboratorSet.size,
     storageBytes,
   });
   ```

6. Wrap in try/catch — return 500 with `{ error: 'Failed to fetch stats' }` on failure.

Imports needed: `NextRequest`, `NextResponse` from 'next/server'; `getAuthenticatedUser` from '@/lib/auth-helpers'; `getAdminDb` from '@/lib/firebase-admin'.

```typescript
interface DashboardStats {
  projectCount: number;
  assetCount: number;
  collaboratorCount: number;
  storageBytes: number;
}
const [stats, setStats] = useState<DashboardStats | null>(null);
const [statsLoading, setStatsLoading] = useState(true);

useEffect(() => {
  getIdToken().then(token =>
    fetch('/api/stats', { headers: { Authorization: `Bearer ${token}` } })
      .then(r => r.json())
      .then(data => { setStats(data); setStatsLoading(false); })
      .catch(() => setStatsLoading(false))
  );
}, []);
```
