# Plan 36-01: Add title tooltip to filename cell in AssetListView

## Goal
Full filename is readable in list view on hover — currently truncated with no tooltip.

## Context
- **File**: `src/components/files/AssetListView.tsx`
- **Target**: Line 428 — the name `<span>` inside the Name `<td>`
- **Current**: `<span className="font-medium text-white truncate block max-w-[240px]">{asset.name}</span>`
- **Fix**: Add `title={asset.name}` — native browser tooltip shows full name on hover

## Implementation

### Step 1 — Add title attribute to name span

Change line 428:
```tsx
<span className="font-medium text-white truncate block max-w-[240px]">{asset.name}</span>
```
To:
```tsx
<span className="font-medium text-white truncate block max-w-[240px]" title={asset.name}>{asset.name}</span>
```

## Verification

1. Run `npm run build` — zero TypeScript errors
2. In list view, hover a long filename — full name appears in native browser tooltip
3. Short filenames show no overflow
