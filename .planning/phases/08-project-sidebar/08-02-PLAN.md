Update `src/components/layout/AppShell.tsx` to persist `sidebarOpen` in localStorage.

Changes:
1. Replace `useState(true)` with a lazy initializer that reads from localStorage:
```tsx
const [sidebarOpen, setSidebarOpen] = useState<boolean>(() => {
  if (typeof window === 'undefined') return true;
  const stored = localStorage.getItem('sidebar-open');
  return stored === null ? true : stored === 'true';
});
```

2. Add a useEffect that writes to localStorage whenever sidebarOpen changes:
```tsx
useEffect(() => {
  localStorage.setItem('sidebar-open', String(sidebarOpen));
}, [sidebarOpen]);
```
