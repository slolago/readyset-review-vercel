# Plan 24-01: Safe Zones Opacity Slider

## Objective

Add an `opacity` prop to `SafeZonesOverlay` and an opacity slider control to `VideoPlayer` controls that is visible only when a safe zone is active. Dragging the slider changes overlay transparency immediately. Opacity resets to 100% when toggling a safe zone off or switching zones.

## Context

- `src/components/viewer/SafeZonesOverlay.tsx` — simple `<img>` overlay, needs `opacity` style prop
- `src/components/viewer/VideoPlayer.tsx` — manages `activeSafeZone` state, renders `<SafeZonesOverlay>`, renders `<SafeZoneSelector>` in controls row; opacity slider goes next to the safe zones selector
- `src/components/viewer/SafeZoneSelector.tsx` — read-only for this plan

## Tasks

### Task 1: Add opacity prop to SafeZonesOverlay
**Type:** auto
**Done when:** `SafeZonesOverlay` accepts an optional `opacity: number` prop (0–1, default 1) and applies it via inline `style={{ opacity }}` on the `<img>` element.

**Implementation:**
- Add `opacity?: number` to `SafeZonesOverlayProps` interface (default to 1)
- Pass `opacity` to the `<img>` style object

### Task 2: Add safeZoneOpacity state and slider to VideoPlayer
**Type:** auto
**Done when:**
1. `VideoPlayer` has `safeZoneOpacity` state (default 1)
2. A horizontal range slider (`input[type=range]`) renders in the controls row immediately after `<SafeZoneSelector>`, **only when** `activeSafeZone` is not null
3. Slider range is 0–1, step 0.05, styled consistently with the volume slider (purple fill)
4. `SafeZonesOverlay` receives `opacity={safeZoneOpacity}`
5. When `setActiveSafeZone` is called (zone selected or deselected), `safeZoneOpacity` resets to 1

**Implementation:**
- Add `const [safeZoneOpacity, setSafeZoneOpacity] = useState(1);`
- Wrap `setActiveSafeZone` calls so they also reset opacity to 1
- Add the slider element conditionally on `activeSafeZone !== null`
- Thread `opacity={safeZoneOpacity}` into `<SafeZonesOverlay>`

## Verification

1. With no safe zone selected: opacity slider is not visible
2. Select any safe zone: slider appears next to the Safe Zones selector, overlay is visible at 100% opacity
3. Drag slider to ~50%: overlay becomes semi-transparent immediately
4. Switch to a different safe zone: opacity resets to 100%
5. Deselect safe zone (click "None"): slider disappears; re-selecting shows it at 100%

## Success Criteria

- [ ] `SafeZonesOverlay` accepts and applies `opacity` prop
- [ ] Slider visible only when a safe zone is active
- [ ] Dragging slider changes overlay transparency in real-time
- [ ] Opacity resets to 100% on zone switch or deselect
- [ ] No regressions in volume slider, annotation mode, or other controls
