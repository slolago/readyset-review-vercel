  <name>Task 1: Fix FPS off-by-one in rVFC frame counter (BUG-01)</name>
  <files>src/hooks/useAssets.ts</files>

  <read_first>

  <action>
The bug is in `extractVideoMetadata` inside `useAssets.ts`. The `onFrame` callback is called by `requestVideoFrameCallback`. On the very first invocation `startMediaTime` is null, so it sets `startMediaTime = metadata.mediaTime` AND increments `frameCount` to 1 — but no time has elapsed yet. On every subsequent callback it increments again. After ~1 second of playback at 30fps you have ~30 new frames elapsed (elapsed ≈ 1.0s) but `frameCount = 31` because the initial frame was also counted. This produces `31 / 1.0 = 31fps`.

Fix: On the first callback, record `startMediaTime` and return immediately without incrementing `frameCount`. Only start counting from the second callback onward.

Change this block (lines 102–115):

```typescript
// BEFORE (buggy — counts first frame)
const onFrame = (_now: DOMHighResTimeStamp, metadata: { mediaTime: number }) => {
  if (startMediaTime === null) startMediaTime = metadata.mediaTime;
  frameCount++;
  const elapsed = metadata.mediaTime - startMediaTime;
  if (elapsed >= 1.0 || frameCount >= 120) {
    const rawFps = elapsed > 0 ? frameCount / elapsed : undefined;
    const fps = rawFps !== undefined ? snapToStandardFps(rawFps) : undefined;
    done({ ...base, frameRate: fps });
    return;
  }
  (video as any).requestVideoFrameCallback(onFrame);
};
```

To:

```typescript
// AFTER (correct — first callback anchors time, not counted)
const onFrame = (_now: DOMHighResTimeStamp, metadata: { mediaTime: number }) => {
  if (startMediaTime === null) {
    startMediaTime = metadata.mediaTime;
    (video as any).requestVideoFrameCallback(onFrame);
    return;
  }
  frameCount++;
  const elapsed = metadata.mediaTime - startMediaTime;
  if (elapsed >= 1.0 || frameCount >= 120) {
    const rawFps = elapsed > 0 ? frameCount / elapsed : undefined;
    const fps = rawFps !== undefined ? snapToStandardFps(rawFps) : undefined;
    done({ ...base, frameRate: fps });
    return;
  }
  (video as any).requestVideoFrameCallback(onFrame);
};
```

No other changes to this file.
  </action>

function formatVersionDate(ts: unknown): string {
  if (!ts) return '';
  try {
    const date =
      typeof (ts as { toDate?: () => Date }).toDate === 'function'
        ? (ts as { toDate: () => Date }).toDate()
        : new Date(((ts as { seconds: number }).seconds ?? 0) * 1000);

      day: 'numeric',
      hour: 'numeric',

  }
}
