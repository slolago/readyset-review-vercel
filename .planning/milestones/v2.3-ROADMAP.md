## Expected impact

- Asset viewer: video shows thumbnail poster within ~100ms; `<video>` no longer downloads entire file upfront; comments don't block video interactivity
- Annotation mode: first click opens Fabric canvas in <50ms (pre-warmed)
- Comment submit: optimistic insert makes new comments appear in ~50ms instead of 100‚Äì300ms
- Admin panel: projects tab is instant (eager fetch); routes scale past 500+ records without OOM
- Folder browser: project root load is `max(project, folders)` latency instead of sum; rename state changes no longer re-render breadcrumb/header/unrelated cards
- Bundle: heavy modals (~100KB across 5 modals) no longer ship in initial route bundle; `lucide-react` icons split per-route; Google Fonts non-blocking
- Network: preconnect saves ~100ms TLS handshake on first GCS/Firestore request; `/api/assets` edge-cacheable for 5 min with stale-while-revalidate
